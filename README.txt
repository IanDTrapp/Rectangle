Description
--------------------
This program reads in the coordinates of a rectangle and performs the following operations:
Validate: See if its a valid rectangle
Area: Calculates and prints the area
Rotate: Rotates the coordinates 90 degrees clockwise
Print: Prints the coordinates of the rectangle


Instructions
--------------------
Commands to run:

make run: compiles and runs the program
make compile: compiles the program but doesn't remove it
make clean: removes the executable file
